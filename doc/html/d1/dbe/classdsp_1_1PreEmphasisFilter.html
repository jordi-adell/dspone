Detailed Description

The PreEmphasisFilter class This can be used as a premphasis filter to pre-process speech samples. This is a common action to use a IIR filter to do so. The IIR filter used here is H(z)=1 - a*z^(-1) I can be expressed also as: y[n] = x[n] - a * y[n-1] The value of a i the factor passes to the constructor.
