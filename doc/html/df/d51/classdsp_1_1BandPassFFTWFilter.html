Detailed Description

implementation of a band pass filter using the spectrum weighting technique. By default uses a triangular shape with 1 as maxium gain. If the central frequency is supplied the triangle is assymetric. I filters buffers of 2^order samples. This is due to the FFT algorithm restrictions.
