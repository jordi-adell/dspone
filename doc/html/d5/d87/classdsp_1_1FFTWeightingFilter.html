Detailed Description

This class filters a signal by calculating the FFT of the buffer, multiplies the spectrum by the vector of coeficients provided and then the output signal is obtain by obtaining the inverse FFT of the modified spectrum.

To construct the filter you need to provide the coeficients. The length of the coeficients vector needs to be half of the FFT length. This is because for real signal the FFT is symetric.
