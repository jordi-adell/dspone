Detailed Description

template<class T_Weight, class T_Particle>
class dsp::ObservationModel< T_Weight, T_Particle >

The ObservationModel class Dummy class for testing purposes. The observation model depens on the taks to be performed and needs measurement of sensors to estimage the weights of the particles. This class sets the weight of all the samples to the same weight 1/size.
